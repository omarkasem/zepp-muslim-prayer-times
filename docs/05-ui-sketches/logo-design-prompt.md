# Logo / App Icon — Design Prompts

App icon for **Prayer Times**, a free Muslim prayer-times app for Zepp OS / Amazfit smartwatches.
The icon appears as a **store thumbnail** and **on the watch** (masked to a circle), so it must read
clearly at tiny sizes.

**How to use:** each numbered prompt below is a different visual concept and is fully self-contained —
just copy one block and send it as-is to any AI image generator (Midjourney, DALL·E / ChatGPT,
Ideogram, Gemini, Stitch, etc.). Send different ones to different tools and compare.

---

## Prompt 1 — Minimal Crescent

```text
A flat vector app icon for a Muslim prayer-times smartwatch app (Amazfit / Zepp OS). Concept: a single
clean crescent moon, emerald green (#4edea3) with a subtle deeper-green gradient (#10b981), with a faint
warm-gold (#ffb95f) rim-light along the crescent's inner edge, centered on a deep black (#000000)
circular background. Style: ultra-minimal, modern, geometric, perfectly symmetrical, flat design, crisp
edges, high contrast. No text or lettering. No human figures or faces. Square 1024x1024, single emblem
centered with generous padding so nothing important is near the corners (it will be masked to a circle
on the watch). Must stay recognizable at 48px. Calm, respectful, dignified.
```
